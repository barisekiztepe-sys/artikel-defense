# 

\# 🕹️ Artikel Defense



A retro cyberpunk shooter game for learning German articles (der, die, das),

plurals, and verb conjugations — built because flashcard apps were too boring.



\## Why I built this

Moving to Switzerland meant learning German fast. Instead of Duolingo,

I built a game where falling words shoot back.



\## Features

\- 6 game modes: Story, Endless, Vocab Match, Plurals, Verbs, Revenge Mode

\- Revenge Mode tracks your mistakes and drills you on them specifically

\- In-game shop with ZAP credits, shields, and time-warp power-ups

\- Global Top 10 leaderboard via Firebase

\- Daily quests and streak system

\- Collectible cards unlocked by achievements

\- Built-in audio engine (Web Audio API — no external sound files)



\## Tech Stack

React 19 · TypeScript · Firebase · Tailwind CSS · Framer Motion · Vite

AI-assisted development via Google AI Studio



\## Run Locally

1\. `npm install`

2\. Add your Firebase config to `src/firebase.ts`

3\. `npm run dev`
